## Accessibility & UX Best Practices [app.uxcel](https://app.uxcel.com/courses/common-patterns/booking-best-practices-107)

**Color & Contrast**: [companionlink](https://www.companionlink.com/blog/2024/06/admin-dashboards-how-to-create-a-good-one/)
- WCAG AA compliant contrast ratios (4.5:1 minimum)
- Don't rely on color alone (use icons + text)
- Dark mode support with system preference detection

**Keyboard Navigation**:
- Tab through all interactive elements
- Enter to submit forms
- Escape to close modals
- Arrow keys for calendar navigation

**Screen Reader Support**:
- Semantic HTML (`<nav>`, `<main>`, `<section>`)
- ARIA labels for icon-only buttons
- Live regions for status updates
- Skip to main content link

**Error Handling**: [app.uxcel](https://app.uxcel.com/courses/common-patterns/booking-best-practices-107)
- Inline validation with clear messages
- Prevent form submission until valid
- Highlight error fields with red border
- Scroll to first error on submit
- Clear error messages ("Email is required" not "Invalid input")

**Loading & Empty States**: [aufaitux](https://www.aufaitux.com/blog/dashboard-design-principles/)
- Never show blank screens
- Skeleton loaders matching content structure
- Meaningful empty states with CTAs
- Retry buttons for failed requests

## Performance Optimization

**Code Splitting**:
```typescript
const MachinesPage = lazy(() => import('./pages/MachinesPage'));
const AdminPanel = lazy(() => import('./pages/AdminPanel'));
```

**Image Optimization**:
- Lazy load machine images
- Use WebP format with fallbacks
- Responsive images with `srcset`
- Placeholder blur effect

**React Optimization**:
- `React.memo` for BookingCard, MachineCard
- `useMemo` for filtered/sorted lists
- `useCallback` for event handlers passed to children
- Virtual scrolling for long lists (react-window)

**Data Fetching**: [bricxlabs](https://bricxlabs.com/blogs/calendar-ui-examples)
- React Query for caching
- Prefetch on hover for details
- Optimistic updates for approvals
- Debounce search inputs (300ms)
